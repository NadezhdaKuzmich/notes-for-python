Бібліотека призначена для математичних обчислень на вирішення наукових завдань. Вона надає ряд зручних
інтерфейсів для роботи з векторами та багатовимірними масивами та широко використовується в Data Science.
Для встановлення цієї бібліотеки використовуємо менеджер пакетів pip3 (pip3 install numpy).

Призначення
- N-мірні масиви.
- Математичні операції над векторами та матрицями.
- Лінійна алгебра.

Основним типом даних для роботи з N-вимірними масивами є тип даних ndarray,
який містить у собі такі складові:
- Кількість вимірювань (ndim)
- Форма (shape)
- Кількість всіх елементів масиву (size)
- Тип елементів масиву (dtype)
- Розмір компонентів масиву в байтах (itemsize)
- Значення елементів масиву (data)

Основні типи даних (dtype)
- int(int_:int8,int16,int32,int64,uint8,uint16,uint32,uint64)
- float(float_:float16(1/5/10),float32(1/8/23),float64(1/11/52))
- complex (complex_: complex64 (f32/f32), complex128 (f64/f64))
- bool(bool_)
