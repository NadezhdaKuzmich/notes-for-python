JSON формат також текстовий формат. JSON безпосередньо пов'язаний з мовою JavaScript та представляє
визначення структур об'єктів. Формат підтримує невелику кількість типів даних, що спрощує перевірку
даних на стороні мови програмування. Як складові типи даних JSON підтримує об'єкти та списки.
Стандартна бібліотека мови Python містить модуль `json`, який надає зручні інтерфейси для роботи з
цим форматом.

Формат даних JSON
- текстовий формат.
- неоднорідні дані.
- типи даних.

JSON у Python
- Читання файлів.
- Створення файлів.

Типи даних
1. String
2. Number
3. Boolean
4. Array
5. Object

{
  ”records”: [
    {
      “first_name”: “Dmitry”,
      “last_name”: “Ivanov”,
      “age”: 29,
      “children”: []
    }
  ]
