XML – це текстовий формат. За своєю суттю це DOM документ із певною структурою.
Дані містяться в тегах і теги мають власні імена. Даний формат дуже схожий на HTML-обидва містять теги,
але XML призначений виключно для обміну та зберігання даних. У стандартній бібліотеці мови є модуль `xml`
для роботи з цим форматом. Особливістю цього формату є можливість пошуку за документом, використовуючи
спеціальну мову пошуку XPATH.

Формат даних XML
Текстовий формат.
- неоднорідні дані.
- Теги та атрибути.
- Об'єктна модель документа (DOM).

XML у Python
- Читання файлів.
- Створення файлів.
- Пошук елементів документа (XPATH).

Приклад 1 (типи тегів):
 1. <value>10</value>
 2. <value />

Приклад 2 (xml-документ):
<?xml version="1.0"?>
<data>
  <person pk="20">
     <first_name>Dmitry</first_name>
     <last_name>Petrov</last_name>
     <age>23</age>
     <address />
  </person>
</data>
